# 📚 Gerenciador de Estudantes (JavaScript)

Um simples sistema de gerenciamento de alunos feito em JavaScript, rodando direto no navegador via `prompt` e `alert`.  

Permite cadastrar estudantes, registrar notas, calcular médias, listar aprovados, remover estudantes e até gerar estatísticas da turma.

---

## 🚀 Funcionalidades

- ✅ Adicionar estudantes  
- 📋 Listar estudantes cadastrados  
- ✏️ Registrar notas para cada estudante  
- 📊 Calcular média individual  
- 🟢 Mostrar aprovados (média >= 7)  
- 📈 Estatísticas da turma (média geral, maior e menor média)  
- 🏆 Ordenar estudantes por média (ranking)  
- ❌ Remover estudantes  

---

1. Clone este repositório ou baixe os arquivos.
